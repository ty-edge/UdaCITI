The Web

The Web (or World Wide Web) is a collection of documents and other files. Most of the documents are made using Hypertext Markup Language (HTML).

What is HTML?

HTML stands for Hypertext Markup Language. It's used to define the structure or layout of a web page.

Based on what we've learned thus far, two examples of elements that have optional closing tags are the paragraph <'p'> and list item <'li'> tags.

When you leave off a closing tag, the closing of the tag is implied as the browser knows it wouldn't be possible to open a new tag without first closing the optional tag and thus, will insert the closing tag for you. While this is allowed by the browswer, it is much easier for you and other developers to read and follow your code when tags that can be closed are explicitly closed.
